# Smart Calculator (Python Tkinter)

Cette application est une calculatrice graphique développée en Python
avec Tkinter. Elle permet d’effectuer des calculs arithmétiques de base
via une interface simple et intuitive.


## Fonctionnalités

- Addition, soustraction, multiplication, division

- Interface graphique avec Tkinter

- Gestion intelligente des zéros initiaux

- Gestion des erreurs (division par zéro, syntaxe invalide)

- Bouton de réinitialisation (Clear)



## Technologies utilisées

- Python 3

- Tkinter (GUI)

- Programmation événementielle


## Installation

1. Cloner le dépôt :

   git clone https://github.com/Ledestin-gamavo/smart-calculator.git

2. Se placer dans le dossier :

   cd smart-calculator

3. Lancer l'application :

   python calculator.py


## Utilisation

- Cliquer sur les boutons pour entrer les nombres
- Utiliser les opérateurs pour effectuer des calculs
- Appuyer sur "=" pour afficher le résultat
- Appuyer sur "C" pour réinitialiser la calculatrice
- Vous pouvez aussi changer le thème générale de l'application
